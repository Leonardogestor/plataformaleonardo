Teste de Anamnese Financeira

Para testar o formulário de anamnese:

  1. Abra http://localhost:3000/anamnesis
  2. Faça login na plataforma
  3. Preencha o formulário com:
    • Nome: Teste Usuario
    • Data de nascimento: 01/01/1990
    • Complete todos os 9 passos
    • Clique em "Finalizar Anámnese"

Verifique se:

  • Os campos Nome e Data de Nascimento aparecem no primeiro passo
  • O botão "Finalizar Anámnese" funciona corretamente
  • Os dados são salvos no banco
